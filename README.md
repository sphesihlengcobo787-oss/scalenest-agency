# ScaleNest Agency Website

## Overview

ScaleNest is a professional agency website for a top-performing SEO and Shopify optimization agency. The website showcases services, transparent pricing in South African Rands (ZAR), and includes lead generation capabilities.

**Tagline:** Build. Sell. Scale.

## Features

### 1. **Hero Section**
- Eye-catching headline with tagline
- Key statistics (210+ Shopify stores optimized, R 389M+ revenue generated)
- Call-to-action button for free SEO audit
- Animated logo display

### 2. **Services Section**
- 6 comprehensive service offerings:
  - Technical SEO
  - Content Strategy
  - Link Building
  - Conversion Rate Optimization
  - AI-Powered SEO
  - Shopify Optimization

### 3. **Why Choose Us Section**
- 6 key differentiators highlighting competitive advantages
- Proven results and transparent pricing messaging

### 4. **Pricing Section**
- Three transparent pricing tiers in ZAR:
  - **Growth Starter:** R 14,950/month (15-20% below industry average)
  - **Scale Pro:** R 31,680/month (Most Popular)
  - **Enterprise Elite:** R 79,200+/month
- Feature comparison across all tiers
- Clear call-to-action for each plan

### 5. **Process Section**
- 4-step proven methodology:
  1. Audit & Analysis
  2. Strategy Development
  3. Implementation
  4. Monitoring & Optimization

### 6. **Contact Section**
- Lead generation form with fields for:
  - Name, Email, Company
  - Shopify Store URL
  - Goals/Message
- Contact information display
- Social media links

### 7. **Navigation & Footer**
- Sticky navigation bar with logo and menu links
- Comprehensive footer with quick links and company info

## Deployment Instructions

### Option 1: Local Testing
1. Open `index.html` in a web browser
2. All functionality works without a server

### Option 2: Web Server Deployment
1. Upload all files to your web hosting provider
2. Ensure the directory structure is maintained
3. Update contact form to connect to your backend email service

### Option 3: Static Site Hosting
- Deploy to Netlify, Vercel, GitHub Pages, or similar services
- Simply upload the entire folder
