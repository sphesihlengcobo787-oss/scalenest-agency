## Part 1: Google Search Console Setup

### Step 1: Access Google Search Console
1. Go to https://search.google.com/search-console
2. Sign in with your Google account (sphesihle.ngcobo787@gmail.com)
3. Click "Start Now" or "Add Property"

### Step 2: Add Your Property
1. Choose **URL prefix** option
2. Enter: `https://scalenest.co.za`
3. Click "Continue"

### Step 3: Verify Domain Ownership
You have multiple verification options. Choose one:

#### Option A: HTML File Upload (Recommended)
1. Download the HTML verification file provided by Google
2. Upload it to your GitHub repository root directory
3. Commit and push to GitHub
4. Click "Verify" in Google Search Console
5. Wait 24-48 hours for verification

#### Option B: HTML Meta Tag
1. Copy the meta tag provided by Google
2. Add it to the `<head>` section of index.html:
   ```html
   <meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE">
   ```
3. Commit and push to GitHub
4. Click "Verify" in Google Search Console

#### Option C: DNS Record
1. Go to your HostAfrica domain settings
2. Add a TXT record with the value provided by Google
3. Click "Verify" in Google Search Console
4. Wait for DNS propagation (24-48 hours)

### Step 4: Submit Sitemap
1. In Google Search Console, go to **Sitemaps** (left menu)
2. Click **Add/Test Sitemap**
3. Enter: `https://scalenest.co.za/sitemap.xml`
4. Click **Submit**

### Step 5: Configure Settings
1. Go to **Settings** (left menu)
2. Set **Preferred domain**: `https://scalenest.co.za/`
3. Set **Crawl rate**: Default (Google will optimize)
4. Enable **Enhanced Search Appearance** features

### Step 6: Monitor Performance
1. Go to **Performance** (left menu)
2. Monitor:
   - Click-through rate (CTR)
   - Impressions
   - Average position
   - Search queries

### Step 7: Check Coverage
1. Go to **Coverage** (left menu)
2. Ensure all pages are indexed
3. Fix any errors or warnings

## Part 9: Troubleshooting

### Common Issues & Solutions

#### Website Not Indexed
- **Problem**: Google hasn't indexed your site
- **Solution**: 
  1. Submit sitemap in Google Search Console
  2. Request indexing for homepage
  3. Check robots.txt allows crawling
  4. Wait 7-14 days

#### Low Rankings
- **Problem**: Not ranking for target keywords
- **Solution**:
  1. Create more content for keywords
  2. Build more backlinks
  3. Improve on-page SEO
  4. Check competitor content
  5. Wait 3-6 months for results

#### Google Business Profile Not Verified
- **Problem**: Verification code not received
- **Solution**:
  1. Check spam folder for postcard
  2. Try phone verification
  3. Request new verification code
  4. Wait 14 days before requesting again
